### 如何使用這個common

- 直接使用

```
cd /your/project/folder/
git clone https://github.com/YunTechAi-FreeSpaceTechnic/common.git
```

- 作為子模𧇿使用

```
cd /your/project/folder/
git submodule add https://github.com/YunTechAi-FreeSpaceTechnic/common.git common
```
